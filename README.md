# generator-ts-np
Yeoman Generator For Building Node Packages With Typescript.

> This package is in Alpha :warning: You are warned. 

![TS-NP generator](./media/logo.png)

## Install
```
npm install -g generator-ts-np
```

## Usage
```
yo ts-np
```
